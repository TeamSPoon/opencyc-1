Changing the Maximum Level to be Graphed
If you check the option "Show comments for terms", then graphed constants are displayed with their #$comment strings, if they have any. Check this option, then try graphing with these parameters:

     Index Argument: 2
     Starting Term: language
     Microtheory: BaseKB
     Binary Predicate: genls
