Accessing The Cyc Web Interface
The best web-browser configuration for using these tutorials is with two windows open, one to display the documentation pages and the other to display the Cyc Web Interface. If that is awkward, perhaps due to a small monitor, you may want to print this module and just use the browser for the Cyc Web Interface.

The current KB browser uses frames and JavaScript, so you should have a browser that supports these to have full functionality. You can use the Cyc Web Interface without frames or JavaScript, but you will lose some functionality.
