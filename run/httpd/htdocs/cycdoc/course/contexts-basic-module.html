This is a basic introduction to contexts in Cyc®. This includes Cyc's ''microtheories'' and their ''Domain Assumptions,'' and various specific kinds of commonly used microtheories. It also introduces the #$genlMt access relation between contexts, and the resulting partial order (ordered set) of contexts. It does not include all of the advanced Cyc operations involving contexts.
