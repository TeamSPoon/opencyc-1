Adding Adjectives
Adding Proper Names
If you want to represent the regular form of an adverb, then choose the "Add regular strings to lexicon" button to add the string. If you would also like to represent the comparative and superlative forms, then choose the "Add irregular strings to lexicon" button.

Note that very few adverbs have comparative and superlative forms. Please contact the CYC® NL team if you have any questions about adverb denotations.
