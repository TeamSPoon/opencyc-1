Some Exercises with #$Cleaning
There may be many assertions in the KB with different second arguments given a predicate and some first argument. For example, two "entries" on the predicate #$isa for #$TeethCleaning are:

     #$TemporalObjectType
     #$DefaultDisjointScriptType

In prefix notation, these would be represented as:

     (#$isa #$TeethCleaning #$TemporalObjectType)
     (#$isa #$TeethCleaning #$DefaultDisjointScriptType)
