Using The Cyc Web Interface
Accessing the Cyc Web Interface
Viewing Cyc Constants with the Web Interface
The Cyc Navigator page is a set of menus of links which provide direct access to most of the Web Interface's facilities. The Cyc Navigator collects all the facilities in one place, so if you don't see what you are looking for on the page you're currently on, go to the Cyc Navigator.

You can select "Cyc Navigator" from the Cyc Main Menu. Also, on most pages presented by the Web Interface, there is a link called "Tools" in the top left corner. From there you can go to the Cyc Navigator. You can also add the Cyc Navigator to your toolbar by clicking in the checkbox and clicking "Update Toolbar".
