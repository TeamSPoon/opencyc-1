Notice that each constant in the graph is preceded by a red diamond. This is a link which will specify a new graph with the same parameters except that it starts at the adjacent constant.

If you had the hierarchy from the previous page, you could click on the diamond (we used a + in this module to represent a red diamond) next to "Snake" and get a new display that looks something like:

  Hierarchical display radiating from Snake
             isa: OrganismClassificationType
           genls: Reptile
         Context: BaseKB
       Predicate: genls
           Index: 2

  [?]  Back   [Change Hierarchy Browser Settings]

     +Snake
	+BoaConstrictor
	+Cobra
        +CoralSnake
	+PitViper
	   +Copperhead
	   +Rattlesnake
	   +WaterMoccasin
	+Python
	+WaterSnake
	   +WaterMoccasin
