This module is a tutorial on using the Cyc Web Interface and some of the common notations used in Cyc.

The Cyc Web Interface is also known as the KB Browser, but in this module, it will often be referred to as "the web interface" or "the interface".

As in other Cyc documentation pages, references to Cyc® constants include the "#$" prefix. But when using the web interface, you can usually omit the prefix, making typing easier. We'll point out the cases where the prefix is required. Note: By the time OpenCyc v1.0 is released, the "#$" prefix will almost never need to be typed (e.g., it will no longer be required in ASKs or in the SubL interactor).

This document has received some editing to remove inaccuracies, but you will likely still see constants mentioned which are not currently part of the OpenCyc release. However, these constants should get the point across. If this is not the case, or if serious inaccuracies are encountered, send email to opencyc-doc@cyc.com. The Overview page is the place to start.
